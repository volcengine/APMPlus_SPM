# APMPlus_iOS_SPM

这个库是 [APMPlus](https://www.volcengine.com/docs/6431/68850) 的 SPM 版本。 

## 接入方法

### 1.打开 Xcode 的 Package Dependencies，并点击添加按钮。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_06242b259d0d74e231cdefc657af49d4.png)

### 2.从 [工程地址](https://github.com/volcengine/APMPlus_SPM) 中找到 APMPlus SPM 版本的 git 地址：git@github.com:volcengine/APMPlus_SPM.git。然后，在 Xcode 中检索并添加需要的库。 
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_54a6f639e52505fe80bda8850c0f995c.png)

> 说明
> 在 SPM 中为了防止库名冲突，因此，在 CocoaPods 的 subspec 名称之上增加了  APMPlus_ 前缀。比如：我们想了解 library 名为 APMPlus_Disk的功能，我们可以取出前缀，在这个[文档](https://www.volcengine.com/docs/6431/68850#f391c95d)中搜索 Disk 的功能说明。

### 3.接入方法
该方案的接入方式和 [CocoaPods 的接入方式](https://www.volcengine.com/docs/6431/68850#878f6178) 基本相同，需要在导入头文件时，修改为#import <APMPlus_Public/APMPlus_Public.h>即可。
